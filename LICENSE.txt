Hak Cipta Dwi Bakti N Dev dan Kelompok Yazri

Semua hak cipta dilindungi. Anda tidak diperbolehkan untuk menghapus kredit penulis atau atribusi tanpa izin tertulis dari Dwi Bakti N Dev dan Kelompok Yazri.

Ketentuan Penggunaan:

1. Anda diperbolehkan menggunakan untuk keperluan pribadi dan komersial.
2. Anda diperbolehkan memodifikasi atau menyesuaikan sesuai kebutuhan.
3. Anda diperbolehkan mengonversi untuk digunakan pada CMS apa pun.
4. Anda diperbolehkan membagikan atau mendistribusikan dengan mencantumkan nama Dwi Bakti N Dev dan Kelompok Yazri.
5. Anda diperbolehkan menampilkan tangkapan layar atau tautan di blog atau situs web lainnya.

Larangan:

1. Anda tidak diperbolehkan menghapus kredit penulis atau atribusi tanpa izin tertulis.
2. Anda tidak diperbolehkan menjual, menyewakan, melisensikan, atau mendistribusikan ulang tanpa izin.
3. Anda tidak diperbolehkan mengunggah ke situs web template atau koleksi template tanpa izin.

Lisensi ini dapat dibatalkan jika Anda melanggar salah satu ketentuan di atas.

Silakan hubungi kami jika Anda memiliki pertanyaan lebih lanjut.
